# Simple Go boilerplate

Contains self-sufficient build system, document-generator, formatter & linter.


## Usage

```
./scripts/build
./scripts/doc
./scripts/fmt
./scripts/install
./scripts/lint
./scripts/run
./go-boilerplate -c|--config ./config.yaml
```
